# 🍴 Forkify — Recipe Finder App

A recipe search application that consumes a **REST API** using Fetch & async/await.

## 🔗 Live Demo
👉 [View Live](https://abdelfattah8.github.io/)

## ✨ Features
- ✅ Search thousands of recipes via REST API
- ✅ Async/await with proper error handling
- ✅ Loading states and empty-state feedback
- ✅ Enter-key search support
- ✅ Quick-tag filters (Pizza, Pasta, Burger…)
- ✅ Full ingredient list in Bootstrap modal
- ✅ Animated staggered card reveals
- ✅ Responsive design

## 🛠️ Built With
- JavaScript (ES6+)
- Fetch API / async-await
- Bootstrap 5
- Forkify API
- HTML5 & CSS3

## 🚀 How to Run
1. Clone the repository:
   git clone https://github.com/abdelfattah8/Forkify.git
2. Open index.html in your browser
